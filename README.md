# 鸣潮·爱弥斯专题网站

这是一个为《鸣潮》3.1新角色爱弥斯制作的专题网站。

## 特性

- **响应式设计**: 适配桌面、平板和手机。
- **现代技术栈**: 使用 ES6 Modules, CSS Grid/Flexbox。
- **性能优化**: 代码分割，懒加载 (可扩展)。
- **视觉效果**: 粒子动画，毛玻璃效果，平滑过渡。
- **交互体验**: 导航菜单，灯箱画廊，加载动画。

## 安装与使用

1.  将所有文件放置于你的 Web 服务器目录下。
2.  确保 `assets/images/` 文件夹包含你提供的图片文件。
3.  访问 `index.html`。
